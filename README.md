# Elctrm Chckr

 * Installed electrum (using pip)
 * Generated wallet (see electrum docs)
 * Electrum daemon started with status - **connected**
```javascript
electrum daemon status
{
    ................., 
    "connected": true,
    .................,
    .................,
}

```

 * Python 2.7

## Usage

```bash
python elctrm.py -h
```
